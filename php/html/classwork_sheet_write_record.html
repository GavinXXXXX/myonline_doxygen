How to use ... create new class extends from excelReport in this file in construct method,call parent::__construct([ Your Excel Template File Name ],[ If Use Excel ,Set it to TRUE ]); then after call parent::__construct, create array like this: $param[self::EI_START_COL] = "[ The Column Where Started ]"; $param[self::EI_START_ROW] = [ The Row number Where End ]; $param[self::EI_END_COL] = "[ The Column Where End ]"; $param[self::EI_SAVE_FILE_NAME] = 'File Name Which To Be Saved'; $param[self::EI_TITLE] = '[Optional,Can Ignore]'; $param[self::EI_TITLE_COORD] = "[Optional,Can Ignore]"; $param[self::EI_COMPAT_SERIAL_COL] = "[Optional,Can Ignore]"; $param[self::EI_CATEGORY_NAME_COORD] = "[Optional,Can Ignore]"; call $this->setExcelInfo( [ Your Excel Template File Name ],$param ); Caution: EXCEL_TEMPLATE."/".[ Your Excel Template File Name ] Must Exist,Otherwise It Will Not Work

then how to write value into your template File... first call $this->prepareNewSheet(); then assign a variable with $this->worksheet,it will create new worksheet copy from template call $this->setCv( [ variable with $this->worksheet ],"[Column Identifier]",$this->ws_nextRow,"[Cell Value]" ) finally set workSheet Title By Using following procedure... $this->ws_title = "Your workSheet Title Name" // SAVE THE FILE,MAKE SURE TMP_DIR HAS DEFINED,AND YOU HAVE RIGHT TO ACCESS THIS DIRECTORY DEFINED IN TEMP_DIR $this->save();
