Detailed Description

DocusignForm implements functionality of Send Documents To Specified recipients via DocuSign web service
,it also has mechanism about fetch envelope status , save already signed documents , and update these information
to signedDocumentEnvelope and signedDocumentRecipients table Useage:

  1. $o = new DocusignForm();
  2. $o->newEnvelope(...);
  3. [optional] $o->setExpiration();
  4. [optional] $o->setAutoNavigation();
  5. [optional] $o->setReminders();
  6. $o->pushDocument(...);
  7. $o->pushRecipient(...);
  8. $o->pushTab(...);
  9. $o->pushAttachment(...);
  10. $o->sendNow(...);
  11. $evnelopeId = $o->getLastEnvelopeId(); // if sendNow() return true
