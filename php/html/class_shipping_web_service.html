get_clean_microtimestamp_string ( )

Generates and returns a string of digits representing the time of the current system in microseconds granularity.

Compared to the standard time() function, the microtime() function is more accurate and in addition, successive quick calls inside a loop generate unique results; which can be quite useful in certain cases.

Our function below generates digits only output based on the time stamp generated by the microtime() function.
